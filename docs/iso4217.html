The ISO standard ISO 4217:1995, from the technical committee TC 68, contains codes for the representation of currencies and funds. This page contains a list with 184 currencies and thier codes according to this standars. Each currency code is a three letter ID. Every position in the ID is from the A-Z character set. No guarantees that the following list contains all codes, nor that those present are correct.
